# PDF Generator Node

Микросервис **PDF Generator Node** отвечает за формирование и рендеринг итоговых PDF-отчетов участников. Сервис интегрируется с централизованным хранилищем (Storage API) для получения HTML-шаблонов и сервисом подписания (Signer API) для наложения криптографической ЭЦП.

## Архитектура

Сервис построен на базе **FastAPI** и не хранит собственного состояния и файлов, а выступает в роли связующего звена для обработки данных.

## Конфигурация

Настройка сервиса осуществляется через переменные окружения или файл `.env`:

| Переменная | Описание | Значение по умолчанию |
| --- | --- | --- |
| `PROJECT_NAME` | Название сервиса | `PDF Generator Node` |
| `STORAGE_API_URL` | Базовый URL сервиса хранилища | `http://storage` |
| `SIGNER_API_URL` | Базовый URL сервиса подписания | `http://signer` |
| `STAMP_TEXT` | Имя ассета текста печати в хранилище | `stamp_text` |
| `STAMP_IMG` | Имя ассета изображения печати в хранилище | `stamp` |

## API Справочник

### Системные эндпоинты

* **`GET /health`**: Проверка работоспособности сервиса.

### Генерация отчетов

* **`POST /api/v1/generate`**: Принимает JSON с результатами игрока, рендерит HTML, конвертирует в PDF и отправляет на криптографическую подпись.

    * **Тело запроса**: `CTFdReportData` (данные игрока, команды и решенных тасков).
    * **Ответ**: Сгенерированный и подписанный PDF-документ (Base64).

## Разработка и тестирование

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск сервиса

```bash
uvicorn src.main:app --host 0.0.0.0 --port 8000
```

### Запуск тестов

Для проверки логики макросов, клиентов и API используется `pytest`.

```bash
pip install -r requirements-dev.txt
pytest tests/
```

## Развертывание в Docker

### Сборка образа

```bash
docker build -t ctf-pdf-gen .
```

### Запуск контейнера

Для корректной работы сервиса необходимо передать URL-адреса смежных микросервисов через переменные окружения.

```bash
docker run -d \
  --name pdf-gen \
  -p 8001:8000 \
  -e STORAGE_API_URL="http://storage:8000" \
  -e SIGNER_API_URL="http://signer:8000" \
  ctf-pdf-gen
```
